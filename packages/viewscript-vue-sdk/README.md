# viewscript-vue-sdk

> An SDK for using Viewscript with Vue/Vuetify

## <a name="install"></a>Install
```bash
$ npm install viewscript-vue-sdk --save
```

## <a name="test"></a>Testing

```bash
$ npm test
```
